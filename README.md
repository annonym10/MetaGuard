# MetaGuard

Приложение для защиты конфиденциальности путем удаления метаданных из файлов.

## Функции

- **Анализ метаданных** в JPEG, PNG, PDF, DOC, DOCX файлах
- **Оценка рисков** утечки конфиденциальной информации
- **Удаление метаданных** с сохранением работоспособности файлов
- **Локальная обработка** - данные никуда не отправляются
- **Пакетная обработка** до 100 файлов одновременно

## Установка

### Для пользователей
Скачайте установщик для вашей ОС из раздела [Releases](#).

### Для разработчиков
```bash
# Клонируйте репозиторий
git clone https://github.com/annonym10/MetaGuard.git
cd metaguard

# Установите зависимости
npm install

# Запустите приложение
npm start
